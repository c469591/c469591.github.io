小白也能用hugo架設自己的網站-1 相關工具的下載與安裝

這次要介紹的是一套名為hugo的架站軟體，用這套軟體，小白也能輕鬆的製作出自己的網站，
只要你會複製、貼上，以及打簡單的幾個英文字母，就可以輕鬆的控制網站呈現出你期望的模樣喔!
不用學寫網頁，只要會用MarkDown寫文章就可以完成，
讓我們馬上開始吧!

hugo簡單介紹

hugo是一套靜態網站架設工具，你需要做的就是下一條指令，hugo就會建立一個專案資料夾，
這個資料夾包括網站所有的設定檔案，你只要在github找一個hugo的網站主題，像穿衣服一樣給它套上去，
然後新增文章，文章內容可以用MarkDown語法來撰寫，不需要用html語法，
最後再下一條指令，hugo就會把你的文章、主題模板、配置等等，生成一個名為public的網站資料夾，
網站資料夾就可以放到server裡，這樣其他人就能看到你的網站了，
整個過程中，你只要會一點MarkDown語法，會複製、貼上，以及會打中英文，就可以成功的製作自己的網站，不需要你會寫專業的程式碼，
另外要補充的是，我也是剛接觸網站架設不久，對於各種指令跟網站也是剛開始摸索，
所以有任何錯漏，也歡迎指正或討論囉!

我的系統環境概述

為了給讀者一個可以參考的系統環境，簡單說明一下寫這篇文章時我的環境

  • windows10 21H2
  • nvda2021.3.5

步驟

  1. 下載、安裝所需的套件
  2. 建立網站專案的資料夾
  3. 下載網站主題模板
  4. 配置網站的設定檔案
  5. 測試網站運作是否正常
  6. 新增文章
  7. 部署到自己的linux主機、或是利用github來架設

安裝git工具

git是一套資料版本管理工具，不過我們所需要的只是透過這個工具來下載網站主題，
當然，你也可以將自己的專案加入git的版本管理中，不過本文會暫時略過這部分，
有興趣深入研究可以參考特種兵在討論區寫的git教學文章
特種兵的git教學

  1. 接下來，下載git工具→
    點我下載git
  2. 打開下載好的git安裝檔，目前我下載的版本是
    Git-2.32.0.2-64-bit.exe
  3. 安裝過程基本上可以全部按next，然後按install，最後按finish
  4. 這樣就安裝好git工具了，git的相關指令也會被加入到環境變數中，
    環境變數是讓我們可以在cmd直接下載網站主題模板的關鍵喔!所以必不可少。

安裝Chocolatey(巧克力)

Chocolatey(巧克力)是一套軟體管理器，我們可以利用這個工具來下載軟體，
不過能下載的軟體僅限於通過巧克力官方審核的軟體，並非任何軟體都有，
我們需要透過巧克力來安裝hugo這套工具，這也是hugo官方建議的安裝方式之一，
關於巧克力支援的軟體列表，可以參考這裡→
點我看巧克力官方的軟體支援頁面

  1. 按鍵盤上的
    win鍵+r
    輸入
    cmd
    按enter
  2. 將下方安裝巧克力的指令複製，注意只有一行，但很長，
    你可以在開頭按nvda+f9，在結尾處按nvda+兩次f10來複製整段文字，指令如下→
    @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"
  3. 找到剛剛開啟的cmd視窗，視窗名稱通常是
    系統管理員: C:\Windows\system32\cmd.exe
    接著按ctrl+v貼上剛剛複製的指令，然後按enter
  4. 注意電腦必須連接到網路，此時就會開始巧克力的下載和安裝，
  5. 要確認安裝是否完成，在cmd的視窗裡按
    nvda+小鍵盤1切到物件檢閱，
    然後按
    shift+小鍵盤9 跳到cmd中的最後一行，
    接著按小鍵盤的7或9，這兩個鍵分別是上一行、下一行的功能，
    安裝成功的話，按7往上看，會看到類似下方的訊息提示
    Chocolatey v0.10.15
    Please run 'choco -?' or 'choco -?' for help menu.
    p.s 意思是已經安裝了巧克力v0.10.15 的版本，後面是巧克力給用戶的一些提示，
    這樣就安裝成功了

用巧克力安裝擴展的hugo

完成了上面的安裝之後，就可以開始用巧克力來安裝軟體了，
這邊要特別強調的是，hugo經過我的研究，有兩種版本，
一種可以說是普通版，另一種是有帶擴展功能的版本，我也不知道為什麼要分兩種，
在最一開始還沒搞清楚這個差別時，讓我浪費的大量的時間在研究它，
言歸正傳，我們要安裝的是有擴展功能的版本，
因為必須用這個版本的hugo，才能正常的執行我們之後要使用的主題模板。

  1. 按照上一個標題的步驟打開cmd的視窗
  2. 複製下面這行指令→
    choco install hugo-extended -confirm
  3. 找到cmd的視窗並ctrl+v貼上，接著按enter
  4. 接下來就會開始安裝了，跟上個標題的操作一樣，按7或9，看看有沒有看到下面的成功訊息
    Chocolatey installed 1/1 packages.
    See the log for details (C:\ProgramData\chocolatey\logs\chocolatey.log).
    如果你看到了這個訊息，那代表擴展版本的hugo也安裝成功囉!
