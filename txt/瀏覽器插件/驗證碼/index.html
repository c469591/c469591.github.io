瀏覽器遇到驗證碼的解決方案，Anti Captcha

我們在使用電腦時，在很多網頁常會遇到要輸入驗證碼才能繼續的狀況，
以前只能請明眼人幫忙，或是用大陸推出的人工視圖的方式來看驗證碼，
這次要跟各位分享Anti Captcha，這是一個瀏覽器上的附加元件，支援firefox跟chrome，
以後再遇到驗證碼，只要按一個快速鍵，花不到一分鐘的時間，就能自動幫你將驗證碼填入編輯框，方便又快速，
不過這麼好的服務視需要付費的，但不貴，最便宜的方案約400台幣有找，
之後的說明會已firefox做示範，chrome應該也差不多。

註冊Anti Captcha

我們要先註冊Anti Captcha的帳號，登入帳號充值Anti Captcha裡的金額，
1美金可以看1000張圖，當金額用完需要繼續充值才能用，現在開始註冊Anti Captcha步驟。

  1. 打開這個網頁
    Anti Captcha
  2. tab找
    Register
    按enter
  3. tab找
    Enter desired login name 編輯區 必要的
    輸入登入名稱，例如我寫lamb
  4. tab找
    Enter your email address 編輯區 必要的
    輸入電子郵件地址，填入可正常收件的email吧
  5. tab找
    Register按鈕
    按enter，之後會發送登入密碼到你的電子郵件
  6. 打開信箱，可以看到來自
    Anti-Captcha.Com 的郵件，
    打開郵件可以在內容看到這些重要的資訊，
    Login:
    Password:
    Secret key:
    請將這些資訊保存好，存成txt放在雲端或外接硬碟，以免丟失，
    這也是我們在附加元件裡用來登入帳號的重要資訊
  7. 回到剛剛Anti Captcha的網頁，如果不小心關了可以回到上文重新打開網頁，
    tab找
    Sign In按鈕
    按enter
  8. tab找這兩個編輯框
    Your login here 編輯區 必要的 可自動完成
    輸入註冊時用的名稱，不是打電子郵件
    Your account password field 編輯區 受保護 必要的
    輸入剛剛從郵件裡存好的密碼
    tab到
    Sign In按鈕
    按enter，如果帳密無誤就能登入成功，
    瀏覽器視窗的標題會變成
    Dashboard - Mozilla Firefox
  9. tab或shift+tab找到在網頁上方的
    Accept User Agreement按鈕
    按enter，網頁的其他內容才會顯示

在firefox上安裝Anti Captcha附加元件

  1. 打開這個網頁，內有firefox跟chrome版
    點我打開附加元件下載頁面
  2. tab找
    Download功能表按鈕
    按enter展開，tab找
    Mozilla Firefox or Tor Browser連結 Installation guide for AntiCaptcha plugin for Mozilla Firefox, Tor Browser
    按enter
    p.s下面的步驟建議先看完再操作，把快速鍵記一下就可以直接裝好
  3. tab找
    Install latest AntiCaptcha plugin for Mozilla Firefox, Tor Browser連結 Install AntiCaptcha plugin for Mozilla Firefox, Tor Browser
    按enter
  4. 聽到以下訊息
    請注意 要允許 antcpt.com 安裝附加元件嗎？
    按alt+c
  5. 聽到以下訊息
    請注意 要安裝 AntiCaptcha automatic captcha solver 嗎？
    按alt+a
  6. 聽到以下訊息
    請注意 AntiCaptcha automatic captcha solver 至 Firefox。
    允許此擴充套件於隱私瀏覽視窗中執行 核取方塊

    ok完成安裝

設定Anti Captcha附加元件

這邊你也可以不用設定，但不設定的結果是沒有音效，且當網頁中有驗證碼就會自動辨識，
以下說明如何開啟音效，以及不讓附加元件自動辨識。

  1. 在firefox裡按alt，往右找工具，往下找附加元件，按enter
  2. 切成瀏覽模式，按nvda+空格，
    按h或shift+h找
    AntiCaptcha automatic captcha solver連結 AntiCaptcha automatic captcha solver 0.50標題 第3級
  3. 往下找
    按鈕 折疊 子功能表更多選項
    按enter，往下找
    選項 4 之 2
    按enter
  4. tab找
    Anti-Captcha.com account key 編輯區 可自動完成 Enter your Anti-Captcha.com account key here
    輸入之前從郵件裡要大家保存好的Secret key:後面的內容，
    32個字元，英文加數字，注意不要複製到空格
  5. tab找
      Play sounds The Plugin plays sound when CAPTCHA found, while solving process and in case of success or error. 核取方塊 沒勾選 Turn on/off sound mode
    按空格勾選
  6. tab找
      Use predefined regular image CAPTCHA marks Regular image CAPTCHA will be automatically found, marked and solved on every web page. It's based on selection of other users, but all your local marks will have priority anyway. Disable if you don't want image CAPTCHAs being solved everywhere they are found. 核取方塊 勾選 Turn On/Off automatic image CAPTCHA selection and solution
    按空格取消勾選
  7. tab找
    Save按鈕 Save changes
    按enter，保存設定，如果上述設定無誤，你能聽到保存設定的音效，
    ok終於大功告成，可以關閉附加元件管理員的頁面了，接下來說明如何使用

如何使用Anti Captcha

只要在需要輸入驗證碼的編輯框按
ctrl+shift+大鍵盤數字6
聽到音效的聲音，此時Anti Captcha就會開始辨識驗證碼，
過程中會有答答答的音效，成功了也會有音效，
檢查一下驗證碼的編輯框，你會發現已經自動填入辨識出的驗證碼，
如果要重新辨識，可以按f5刷新網頁，
根據之前使用的經驗，已經按快速鍵辨識過一次的網頁，如果換了驗證碼的圖，就會自動重新抓圖辨識，除非離開網頁才不會自動辨識，
另一種方式是找到編輯框旁邊Anti Captcha的按鈕，按enter就會取消辨識，
ok，此篇到此結束。
