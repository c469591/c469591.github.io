reaper快速入門-2 錄音和節拍器

上篇我們說完了設定，這篇就來講該如何進行錄音跟節拍器的控制，準備開始。

認識工程和軌道

簡單來說，每個工程就是一個檔案，在reaper當我們儲存工程時，會產生一個副檔名為rpp的檔，這個檔包括了所有的音軌、拍號、調用的媒體文件等等的資訊，
每個工程裡必定都會有軌道，他就像記事本裡的每一行，我們在reaper裡按上或下就是在軌道中跳轉，
而軌道裡可以有很多東西，包括調用進來的媒體文件，例如mp3，或是你錄音進去的聲音，或純粹只在軌道上掛上不同的效果器，或你在midi鍵盤上彈奏的midi事件，
也就是上面說的，他就像記事本的每一行，就看你想在裡面寫什麼，他就會呈現出你想要的模樣，
你可以建立很多軌道，第一軌打鼓、第二軌彈bass、第三軌唱歌，錄製完按空格，三條軌道就會一起播放，
你也可以任意讓某條軌靜音，或讓某條軌獨奏，盡情發揮，就是這麼簡單。
這裡還有我錄製的演示影片，你可以
點我收看演示影片

關於工程、軌道、對象、片段，這些到底是什麼?

上面簡單了解了概念之後，就來更嚴謹的說明工程裡常看到的這些東西，幫助你未來在操作時更有概念。

  1. 工程:
    在台灣，很多音樂人應該都會將工程稱為專案或project，工程只是對岸的翻譯，其實都是同一個東西，
    但為了在學習時更方便，我們就已翻譯包中用的工程來稱呼吧，
    在音樂工作站中，工程是最大的單位，承載了包括軌道、對象、片段等等內容，
    在這裡，我們可以把工程想像成一個大廚房
  2. 軌道:
    軌道用來裝各種媒體對象，包括很多影片格是、音訊格式、midi對象等等，
    在此，我們把軌道想像成一張大長桌
  3. 對象:
    只要是媒體格式，包括影片、音樂檔，丟進reaper裡，就等同一個對象，
    在reaper裡錄音、製作midi，最開始的單位也是一個對象，
    在此，我們把對象想像成一個大碗
  4. 片段:
    片段是裝在對象裡的，
    包括音訊、影片、midi，都是存在對象裡的一個片段，
    另外，一個對象可以放好幾個片段，
    在此，我們把片段想像成大碗裡的白米飯，
    當一個對象裡有很多片段時，就像大碗裡有白米飯、油飯、雞腿等等，
    但要注意的是，一次只能有一個片段在最上面，也就是在一個時間內，只能播放一個片段，
    如果a片段有3分鐘長，b片段只有1分鐘長，那a片段多出來的2分鐘會被裝在另一個對象裡，
    在此，片段想像成各種類型的食物
  5. fx效果
    效果氣就像各種調味料，
    我們可以將效果器放在軌道上，也就像是桌上的所有食物被加了料一樣，
    也可以將效果器放在對象上，就像只在某個碗裡加料一樣，
    在未來當你掌握了更高深的技術，你還能控制在軌道的那些地方料要加多，那些地方料要加少。

播放光標和編輯光標

上面認識了reaper裡的廚房、桌子、碗和實務的概念後，就來認識另一個也非常重要的光標概念吧。
嚴格來講，光標也是對岸的翻譯，台灣應該叫游標，但為了方便還是用光標來稱呼。

  1. 從左至右
    在reaper裡，最左邊是最開頭的點，最右邊是結束的點，
    每次播放時，都會從左至右來播放。
  2. 編輯光標
    編輯光標就像廚師，當我們要對一個時間點的內容作處理時，就需要移動編輯光標到需要編輯的時間點，
    就像廚師要將一個碗內的食物分裝成兩個碗，必須讓廚師走到碗所在的桌子邊，才能處理碗裡面的食物一樣
  3. 播放光標
    播放光標就像食客，從最左邊的桌子開始吃碗裡的食物，直到吃到最右邊沒食物了才會回到廚師所站的地方
  4. 也就是說，在reaper裡，有兩個光標，一個是用來編輯，一個是在播放時會隨著時間的推移逐漸往右，
    播放光標在播放完才會自動回到編輯光標所在的時間點，
    在播放時，你也可以按
    /
    把編輯光標移動到播放光標的時間點

ok，相信在上面的描述中，你應該對整個工作站的脈絡有了更深入的了解。

如何進行標準錄音

會說標準錄音，是因為錄音還有很多種模式，先來介紹標準錄音的方法。

創建軌道

  1. 首先要創建一條軌道，打開reaper之後，在reaper主視窗內按
    ctrl+t
  2. 此時會要你輸入軌道名稱，你可以輸入一個你等一下要做的動作，方便之後管理，例如唱歌
    輸入完後enter，再按上或下動一動，就能看到剛才新增的軌道，像這樣
    1 唱歌 0 个对象
    1 是軌道編號，唱歌是軌道名稱，0 个对象是軌道上的錄音、載入的媒體或midi對象等等的數量，
    如果你按照後面的說明進行錄音，你會發現這裡會變成1 個對象，以此類推

錄音

  1. 按上或下選中要錄音的軌道
  2. 按f7，找到
    錄音準備
    告訴reaper這條軌道準備要錄音了
  3. 按f8找到
    正常监听输入
    打開監聽，上一篇的設定沒有問題的話你會在喇叭或耳機內聽到麥克風收到的聲音
    p.s 再按一次f8會聽到监听输入，這個監聽模式會在我們開始錄音之後將監聽關閉
  4. 按一下大鍵盤上的
    w 跳到軌道開頭
  5. 按大鍵盤上的
    r 開始錄音
  6. 錄音完成後再按一次
    r 來結束錄音
  7. 想重新錄製，不想保留之前的錄音那就按
    ctrl+z 來撤銷
  8. 接下來按照第四步驟操作，就能重新錄音
  9. 錄音完成記得按f7將其切換為取消錄音準備，
  10. 要播放一樣按w跳到開頭，按空格播放，再按一次空格停止
    ok 這樣我們就學會錄音了，超簡單

p.s 如果你錄了好幾次音，但沒按ctrl+z撤銷，那這些錄音可是會一直存在軌道的對象上，成為好多個片段，占用硬碟空間的。
另外如果你錄了很多軌，可以善用f5靜音跟f6獨奏的功能，
如果要讓兩個聲音重疊，可以將兩個對象放在不同的軌道上，當對象所在的時間有蟲和時，就會一起被播放。

軌道和對象的基本操作

之前有說過軌道上的對象有很多種類型，例如插入的媒體、midi對象、錄音之後的聲音檔等等，
如果在軌道上同樣的位置，又重複進行錄音，那這些聲音就會被納入同一個對象裡，我們稱之為片段，
同一時間只能播放一個片段，其他的片段不會消失，只是不做聲，隨時可以切換過去使其發出聲音，
要注意的是，如果在軌道上同個位置插入多個媒體，會優先播放第一個插入的媒體，
如果後面插入的媒體時間比第一個長，才會在第一個媒體播放完後跳到後面插入的媒體播放，
再來說說快速鍵。

  • 上或下 當你移動到某個軌道時，也就是選中了這條軌道
  • ctrl+c 複製選中的軌道
  • ctrl+v 貼上複製的軌道
  • 刪除鍵 刪除軌道
  • 空格鍵 以選中的對象進行播放

以下是軌道上操作對象和片段的快速鍵，你需要先選中軌道，在選擇對象。

  • ctrl+左 選擇軌道上左邊一個對象
  • ctrl+右 選擇軌道上右邊一個對象
  • 刪除鍵 選中對象後按刪除鍵是刪除對象，如果沒選對象就按刪除是會刪除整條軌道的，這點要注意
  • t 跳到所選對象的下一個片段
  • shift+t 跳到所選對象的上一個片段

如何將錄音的作品匯出成mp3或wav或其他音訊檔

在目前的翻譯中，匯出成檔案的功能叫做渲染，我覺得這種翻譯實在很神奇，對錯先不論，只要知道他是幹什麼的就好，以下說明步驟。

  1. 在reaper的主視窗按
    ctrl+alt+r
    或是按alt，之後往下找渲染，按enter
  2. 此時就打開了渲染的視窗，預設會停在編輯框上，在這邊輸入要儲存的檔名
  3. 按shift+tab找到
    格式:下拉式方塊WAV折疊
    在這裡按上或下可選要儲存的音訊格式
  4. 如果你要存wav檔，打一個w，就能看到wav了，直接enter就會開始轉換成wav檔，並儲存到上篇文章我們設定好的渲染資料夾內
  5. 如果你在上個步驟選mp3，在選擇好後按tab可以找到
    下拉式方塊 xxx kbps 折疊
    在這上下可以選擇你要的kbps，簡單來說320音質在mp3內最好，檔案也最大，
    kbps越小那音質越差，檔案也越小，選好直接enter就開始儲存了
  6. 好，不管你存什麼檔，如果存好了你按tab會看到關閉跟返回，如果還在存，你會聽到取消，
    這樣就完成囉，可以分享你的大作給朋友了

如何設定拍速、拍號

預設工程的拍速為120BPM，拍號為4/4拍，
打開節拍器的快速鍵為ctrl+shift+m，再按一次為關閉，只有在播放的時候才會發出聲音，
想調整的話，按照以下步驟

  1. 在reaper視窗上按
    alt+enter 打開工程設置
  2. tab找到
    工程BPM: 編輯區 120.000，
    在這裡輸入數字修改你想要的拍速
  3. 在按tab會依序看到
    拍號 編輯區 4，已選取
    / 編輯區 4，
    在編輯框內輸入你想要的拍號，如果不動就直接enter完成設定
  4. 回到reaper主視窗後，節拍器設定為開啟的狀態下，按空格播放，節拍器就會跟著響起囉
  5. 還有另一種更直覺的方式
    alt+= bpm+1
    alt+- bpm-1
    這邊的=(等於)和-(減號)在倒退鍵的左邊兩顆

用鍵盤敲節拍

要設定拍速還有另一個方法，只要在reaper主視窗按四下h，就能依照你按的速度來設定拍速了，不妨去玩看看。

如何設定預備拍

常覺得按r之後，手來不及回到midi鍵盤上，或還沒抓穩麥克風，那你就需要打開預備拍，
讓reaper在一個小節之後才啟動錄音，你就可以好好深呼吸一下準備開始，這是十分實用的功能。

  1. 在reaper主視窗上按
    shift+alt+m
    就能打開 節拍器和預播設置 的視窗
  2. tab找
    錄音前的計數 核取方塊
    按nvda+除號，使用模擬滑鼠的功能將滑鼠移過去，然後除號單擊，就能將其勾選
  3. tab一次，會看到
    計數小節: 編輯區2，已選取
    意思就是會有兩個小節，拍號為4/4拍的狀況下，也就是八拍，之後才會開啟錄音，在這八拍內無論你如何對著麥克風嘶吼都不會錄進去，
    如果你有修改拍號，那就是你所設定的每小節拍數而定，速度也是取決於前面你對拍速的設定，
    我自己是設1小節就很夠，你也可以跟我一樣把這邊的數字改成1，
  4. 改好數字後，按ESC關閉這個是窗就ok了，不須點確定

如何更改標尺的時間單位

對於許多人來說，用小節的方式來顯示可能會不太直覺，如果你想改為用分秒的方式來查看軌道，你可以按照下面步驟設定，
如果遇到更改後沒有立即生效或顯示方式有點異常的狀況，直接重開reaper即可。

  1. 在reaper的主視窗上按alt，往右找
    查看(V) 子功能表 Alt+v
  2. 往下找
    標尺的時間單位 子功能表
  3. 往右，就能看到
    分:秒
    在這按enter，完成設定
  4. 如果要調整回小節的顯示方式，一樣按照上面的步驟來，在展開標尺的時間單位後，往下找
    小節.節拍
    按enter，設定就回來囉
  5. 要查看設定的結果，只要在主視窗上按w或page up、page down動一動，就會聽到小節或時間的提示了
    ok 完成囉。
  6. 最後，不知道osara何時新增的功能，在標尺為節拍的情況下，
    快速的按兩次
    ctrl+shift+j 可以看編輯光標的所在時間

快速鍵

想要在軌道上快速跳轉到想要的地方，可以透過快速鍵來跳轉，順便把上面說過的快速鍵也整理一下放在這裡，提供查閱。

  • page up 上一個小節
  • page down 下一個小節
  • ctrl+page up 上一拍
  • ctrl+page down 下一拍
  • 左和右方向鍵 調整編輯光標位置
    p.s 每一拍都有0%到99%的位置，可以讓我們更精準地找到想要的地方
  • 小鍵盤數字模式下的加號 提高像素，讓左右移動時兼具更短
  • 小鍵盤數字模式下的減號 降低像素，讓左右移動時間距更大
  • ctrl+j 快速跳到想去的小節
    p.s 預設是1.1.00，也就是第一小節、第一拍0%的位置，你可以修改編輯框裡的數值按enter直接跳過去，
    如果要跳到指定時間，就把.改成:，例如1:20，就是一分二十秒，要精細一點也可以改成像這樣1:20.15之類的
  • ctrl+shift+j 查看目前編輯光標所停留的位置，善用這個快速鍵，你可以知道你現在有沒有停在你想要的位置上
  • alt+enter 打開工程設置，設定拍速或拍號
  • ctrl+shift+m 開關節拍器
  • shift+alt+m 打開節拍器設定視窗
  • w 將編輯光標移動到軌道開頭
  • r 開始或結束錄音
  • 空格鍵 播放或停止播放
  • 上或下 當你移動到某個軌道時，也就是選中了這條軌道
  • ctrl+c 複製選中的軌道
  • ctrl+v 貼上複製的軌道
  • 刪除鍵 刪除軌道
  • ctrl+左 選擇軌道上左邊一個對象
  • ctrl+右 選擇軌道上右邊一個對象
  • 刪除鍵 選中對象後按刪除鍵是刪除對象，如果沒選對象就按刪除是會刪除整條軌道的，這點要注意
  • t 跳到所選對象的下一個片段
  • shift+t 跳到所選對象的上一個片段
  • 斜線 將編輯光標移動到播放光標所在的位置
  • alt+= bpm+1
  • alt+- bpm-1
  • 快速的按兩次 ctrl+shift+j 報讀編輯光標所在時間
  • f12 打開快速鍵說明模式
    p.s 打開了說明模式之後，你可以隨意的按鍵盤上的任意按鍵，他會告訴你按下的按鍵是什麼功能，可以幫助初學者找出忘記的快速鍵
