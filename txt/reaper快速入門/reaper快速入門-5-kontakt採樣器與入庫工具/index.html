reaper快速入門-5 kontakt採樣器與入庫工具

接下來我主要會介紹kontakt以及如何載入音色庫，這是玩midi之前一定要學會的事，否則你只能用windows自帶的軟波表，相信你應該有聽過那種假假的midi音樂吧，是的!就是類似那種音色，
而下篇會讓大家下載的是用真實樂器錄音採樣而製作的樂器音色，每種音色庫都是我花錢花力氣從對岸好不容易弄回來的，且都是經過測試能正常使用，
在開始前，我們要先準備一下，方便我們後續的操作，
你必須按照第一篇的教學設定好midi和vst的相關設定。

另外，有任何疑惑不解之處，你也可以

下載和安裝kontakt採樣器

  1. 先在電腦的c槽或d槽根目錄建立一個名為vst的資料夾，我們以後安裝的所有效果器等等的vst檔案都放在這個資料夾裡，方便管理，
  2. 接下來要下載kontakt採樣器，有了這個之後才能透過kontakt載入音色庫
    點我打開kontakt6.60雲端下載連結
    如果無法下載google雲端，可以改用
    點我從小高羊服務器下載6.60
  3. 找到下載的壓縮檔
    Kv6.6.0Rev2.rar
    按快顯鍵選解壓縮至此
  4. 找到剛剛解壓好的
    Native Instruments Kontakt 6 v6.6.0 WiN Rev2
    資料夾，按enter進入
  5. 找到
    Kontakt Setup PC 6.6.0.exe
    並按enter執行
  6. 安裝程式剛打開可能需要等一下，之後我們可以在視窗裡按tab找到next，
    前面的步驟大概是選擇語言，同意條款，後面有幾個步驟我們要設定一下，所以別直接next到底
  7. 到了其中一個步驟tab會看到
    樹狀檢視Basic Installation
  8. 在這裡按end鍵或用上下找到最下面的
    AAX Plug-In
    按空格取消勾選，不過可能不會報讀只會框一聲，不用管直接找next
  9. 一直到Basic Installation的下一步開始，我們要注意一下，在這一步按tab可以看到
    Change... 按鈕 Alt+c
    用畫面瀏覽看一下視窗上的內容，你會看到這些資訊
    Destination Folder
    Select folder where setup will install files.
    Install Kontakt application to:
    C:\Program Files\Native Instruments\Kontakt
    這裡是選擇kontakt主程式安裝目錄的地方，沒特殊目的的話你可以保持預設，繼續next
    p.s nvda畫面瀏覽，nvda+九宮格7，之後九宮格7或9是跳上下行
  10. 接下來我們按tab會看到有一個
    Change... 按鈕 Alt+c
    vst2的Change按鈕需要重新選擇安裝路徑，
    按tab找Change按鈕，在上面按空格，在彈出的視窗裡選第一步我們建立好的vst資料夾，enter回到安裝畫面，
    確認ok就繼續按next
  11. 接下來的步驟，如果你在前面沒有順利把AAX Plug-In取消勾選才會多出這一步，
    在這一步是選擇AAX32跟64的安裝路徑，我們通常用不到這個，所以可以保持預設繼續next，有沒有裝這個都不影響使用，
    如果你不確定是在選kontakt的vst還是AAX的路徑，只要切到畫面瀏覽，用7或9看到
    AAX-Plug-In Path:
    那這一步就是選AAX的，可以安心的下一步了
  12. 最後安裝完成後直接enter就能關閉安裝視窗
  13. 這樣就安裝好kontakt了
  14. 你可以打開第一步建立的vst資料夾，會發現裡面已經多了一個.dll的檔案了

修補kontakt

kontakt本事需要購買的，這裡我們為了學習因此使用此修補程式，經過修補後才能正常使用，
免責聲明→
本軟體僅供學習研究使用，請勿用魚商業用途，如有任何糾紛皆與本人無關

  1. 打開前面解壓後資料夾內的
    Kontakt 6.6.0 patcher.exe
  2. 使用ocr工具，例如落寞ocr進行識別，並且找到這條識別結果
    Kontakt Patcher
    使用小鍵盤八上方的除號模擬滑鼠左擊，之後按tab找
    Extraire按鈕
    按enter，會彈出幾個視窗，
    完成後焦點會回到6.60的視窗，這樣修補就完成了

掃描vst

通常我在新增vst的時候都還會自己操作讓reaper掃描一下存放vst的資料夾，如此萬無一失，
雖然每次重新打開reaper軟體好像都會自己掃描，不過為了例外狀況，還是說明一下步驟，這個步驟只有在有放新的vst插件時才要操作，
白話點說，這個步驟就是告訴reaper，我們放了心的vst插件了，你看一下，把他收好，以後我要用了你在拿出來，
當你有新的vst插件但reaper裡頭卻找不到也可嘗試此步驟喔。

  1. 打開reaper
  2. 按ctrl+p打開首選項
  3. 樹狀檢視找vst
  4. tab找重新掃描按空格
  5. 掃描期間按tab會沒反應，掃完就能動了，找確定enter完成掃描

標準音色庫的入庫方法

如果你不知道那些音色要入庫那些不用，你就把所有音色庫放在一個大資料夾裡，全部掃一掃，能入庫的就會入庫，不能入庫的他會自己跳過不會影響。

  1. 在前面我們下載並解壓的資料夾內找到
    Kontakt 6.6.0 patcher.exe
  2. 使用你手邊的ocr工具識別一下，例如
    落寞ocr
    在此使用落寞ocr舉例，按win+alt+j識別，並找到
    Add Library Tool
    按小鍵盤8上方的除號模擬滑鼠左擊
  3. 此時會開啟選擇資料夾的視窗，右進左出上下選擇目錄
  4. 選好要入庫的音色資料夾按enter，就會開始掃描
  5. 掃描完成會跳出一個對話框，裡面顯示掃描到幾個音色且成功入庫，如果掃不到可入庫的音色也會顯示出來
  6. 按enter關閉彈出的對話框，之後再按alt+f4關閉輔助工具的視窗
