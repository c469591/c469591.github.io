系統優化篇(part3)

哈囉，我是羔羊，這次要接續上一篇的系統優化part2，讓我們再來繼續遮騰系統的優化吧XD，開始前，建議先看看這篇文章，了解後續的優化有哪些，
點我看歐非先生的系統優化
如果你聰明過人的話，或許看到這篇歐非先生分享的優化內容，就會有所了解，這就是明眼人一開始會做的設定，雖然有一小部分跟我們前幾篇重複，但仍然有些部分是裡面沒有提到的，
像是永遠展開通知區域、簡化工具列、切換系統管理員，在歐非先生的文章裡都沒提到，這是因為這些設定是為了讓nvda更好的操作，
接下來，為了照顧看完歐菲的文章後，還是有看沒懂的同學，下方我就來做幾個重要部分的步驟講解，

Windows 10 安全性、Windows Defender

Windows Defender其實在win7就有了，只不過還沒辦法替代市面上的防毒軟體，自從到了windows10之後，才提升到可以替代市面上防毒軟體的程度，
至於好不好用，不是今天的重點，因為本羊是要教你怎麼把它關掉XD，當然這一步你也可以跳過，提升系統安全性嗎，嘿嘿
但對我來說，Windows Defender只是個會沒經過你同意就直接把檔案殺掉的軟體，時常也有誤砍的狀況，而且還會占用系統資源，對於配置比較低階或老舊的電腦，都是沉重的負擔，
本羊覺得，防毒軟體並不是必須的，要讓電腦不被病毒侵害，應該是要從自己的使用習慣上做檢討，不上一些奇怪的網站、不下載來路不明的軟體、不亂插別人的隨身碟，定期對資料做備份，或最重要的，學會怎麼自己重灌系統，或備份系統，
有這些基本的安全意識，具備這些重灌或備份的技巧，你就可以直接拋棄防毒軟體了，愛怎麼遮騰就怎麼遮騰，只要會重灌又會備份資料，你就利於不敗之地啦，
好了，如果你決定了，那就開始下面的步驟吧，要注意的是版本為1903的windows10，設定的方式跟win10的前幾個版本會有一些不一樣，所以以下步驟已1903為基礎來說明，讓我們馬上開始

步驟開始
1-1 win+i打開設定，按一次tab到列表
1-2 按end鍵或用游標一道更新與安全性，按enter，然後再按一次tab到列表上
1-3 找windows安全性，按enter
1-4 按tab找開啟 安全性，按enter
1-5 按tab找到一個清單，預設是停在首頁，游標上下找病毒與威脅防護，按enter
1-6 用空格鍵或enter關閉 实时保护 和 篡改保护，這是檢體系統的名字，繁體系統的話可能按鈕名字會有些許不同
p.s 因為我已經完全關閉了，所以無法準確寫出所在的位置，或許這兩個功能會在病毒與威脅防護裡面的某個進階選項內，或是完成1-5的步驟後按tab就能找的到，這邊只能靠你們變通一下找找看了XD
這個設定是1903之後才有的新選項，可能是為了降低被病毒串改defender的可能所設計的，包括1809之前的版本沒有這兩個選像
1-7 必須完成前幾個步驟以下的操作才會有效，懶人可以直接下載這個reg黨按，下載完成後解壓縮，在關閉防毒.reg的檔案上按快顯鍵選以系統管理員身分執行，會彈出對話框，選是即可完成，後續步驟就可以省略
點我下載關閉windows defender的reg檔
1-8 手動關閉，之後的步驟區分家用版跟專業版，不同版本設定方法不一樣
專業版
接著按win+r打開執行，在裡面輸入
gpedit.msc
後按enter
1-9 會打開，本機群組原則編輯器的視窗，然後tab到清單列表，依照下方的路徑一層一層的進去
電腦設定 2 之 1
系統管理範本 3 之 3
Windows 元件 8 之 2
Windows Defender 防毒軟體 86 之 18
關閉 Windows Defender 防毒軟體，在這個選項上按enter
按tab找尚未設定(C) 單選紐 勾選 Alt+c，游標上下選擇以啟用，在tab找確定enter

1-10 家用版
按win+r打開執行，輸入cmd，按enter打開命令題是字元視窗
確認視窗標題有Administrator或管理員的字樣，如果沒有，就跳回桌面再按tab到執行中的程式，找到命令題是字元的視窗群組，在上面按快顯鍵找命令題是字元，在按快顯見選擇以系統管理員身分執行，
此時視窗標題有了Administrator或管理員的字樣，然後輸入
reg add "HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWord /d 1
輸入玩後按enter，完成，上方的指令有點常，可以直接用複製的比較不會有打錯的問題
