reaper玩mid檔-上 認識mid檔案、合成器，以及下載mid檔案

在這個系列會跟大家分享如何用reaper玩耍mid檔案，以及如何更換mid檔案中不同樂器的音色ok，讓我們馬上開始~

什麼是mid檔案

mid檔案是有了midi之後，才出現的一種檔案格式，
mid檔裡並不是一段錄製好的聲音，而是記錄了各種midi控制器和音符、速度變化的指令，
而目前的windows系統中，都有一個名為Microsoft GS的合成器，
這個合成器裡有大部分常見樂器的音色，讓我們只需用播放器打開mid檔，透過播放器調用Microsoft GS合成器，就能聽到聲音，
一般的mid檔，一分鐘只要約5到10kb的大小，十分輕巧，
簡單來說，mid扮演的就像樂譜的角色，透過電腦自帶的gs樂手來演奏出音樂。

開啟reaper中的gs合成器

reaper預設情況下，當載入mid檔案後，播放時是不會以gs合成器來播放的，
需要在reaper首選項中開啟gs合成器，這樣才能指定軌道用gs合成器輸出，播放出聲音。

  1. 打開reaper，按ctrl+p打開首選項
  2. 在樹狀檢視找到
    音頻
    按右鍵展開，再往下找
    midi設備
  3. tab找
    列表1 清單 通過右鍵單擊設備名稱並選擇啟用輸出, 啟用要使用的 MIDI 輸出設備.
  4. 往下找
    Microsoft GS Wavetable Synth [小心使用]; <禁用>; 0
    按快顯鍵找
    啟用輸出
    按enter
  5. 注意，這一步可以依需要開啟，
    一樣找到
    Microsoft GS Wavetable Synth [小心使用]; <禁用>; 0
    按快顯鍵往下找
    配置輸出...
    按enter，tab將這兩個選項勾選後按確定
    發送時鐘到此設備 核取方塊
    用低延遲/低精度模式打開設備 核取方塊
  6. shift+tab找確定enter
    ok，設定完成，之後會統一說明如何將軌道以Microsoft GS來播放。

三種mid檔案格式的區別

根據midi掃盲篇的介紹，mid檔有三種格式，midi0、midi1、midi2，
我就以我個人的經驗粗略分享一下，有誤也歡迎指正。

  1. midi0
    單軌多通道，一般我下載的卡拉ok檔案都是這種格式，
    在reaper裡打開的話，只會有一條軌道，這條軌道上還有16個通道，每個通道相當於一種樂器，
    每個通道裡還會記錄一種樂器的音符、以及各種控制器的訊息
  2. midi1
    多軌單通道，簡單來說，當載入到reaper後，每個軌道都只用到一個通道，
    也就是每一個軌道就是不同的樂器，軌道上的midi對象就是該樂器的樂譜
    3 midi2
    這種格式至今為止我還沒搞清楚

p.s 雖然mid檔案格式有點亂，但對於reaper來說，那怕是midi0這種格式，只有單軌多通道，也可以透過reaper的功能將之拆分為多軌單通道，
另外以我的經驗，本來就是多軌單通道的mid檔，編輯者有比較大的機率會對每條軌道做命名，
例如第一軌命名為鋼琴、第二軌命名為弦樂等等，這種檔案對於我們的後續編輯會較為方便，
網路上也有人稱之為mid工程文件，ok以上介紹提供各位參考。
