Reaper Plug Ins系列 5 Auto Tune效果器篇 修掉、電音的參數和使用

 2022.4.29  9 分鐘

reaper plug-ins系列-5-auto tune效果器篇 修掉、電音的參數和使用

這篇延續上篇的內容，要開始介紹auto tune效果器內的參數，以及演示每個參數使用後的效果，
其實控制這些參數並不難，難的反而是靈活的運用這些參數到需要的地方，
因此plug-ins(效果器)，想要用的巧妙，是非常需要多練習、多嘗試來累積經驗的。
接下來我會介紹auto tune的兩個效果器:
用來修調的→VST3: Auto-Tune Pro (Antares)
用來做出電音效果的→VST3: Auto-Tune Access (Antares)
在此要強調電音只是一種音樂風格，並不是一種效果器類型，他的根本還是一顆修調效果器，
只不過Access是我覺得最適合做電音效果的一種，
至於五款修調效果器中的auto key這顆效果器，只是拿來辨識聲音的調性的，到目前為止我還不知道如何用nvda操作。

Auto-Tune Pro修調效果器參數介紹

在軌道上加載Auto-Tune Pro後，按p會有很多參數，以下介紹我比較常用的幾個參數，其他的參數各位也可以自己嘗試看看。
p.s 按p開啟的視窗，從參數的列表裡可以按上、下查看所有可控的參數，tab一次是列表所選參數的值，一樣用上、下調整值，
下面的每個參數名稱後都有一個編號，可以利用這個編號在參數列表中快速找到對應的參數

  1. Correction Mode (1)
    這個參數有兩個值，分別是
    Auto mode跟Graph mode
    預設是Auto mode，也就是自動修音的意思，保持Auto mode即可
  2. Scale (2)
    這是選擇音階類型的，例如MAJOR(大調)或MINOR(小調)，大部分流行歌選擇大調即可，
    上、下動一動還能看到其他的類型，例如CHROMATIC、ARABIC、INDIAN等等，
    你應該知道某些地區他們的音樂聽起來很不一樣，就是因為他們的音階跟自然大小調比起來，可能多了某些音或少了某些音，
    這些音形成了特殊的音階排列，給人與眾不同的聽感，大家最常聽到的中國風音樂，就是由五聲音階組成，如果你還不太懂，建議看看這篇
    點我看reaper 超基礎樂理
    預設為CHROMATIC，在此模式下不管什麼key，都是自動把歪掉的音拉到最靠近的音高上，並沒有不同調性的差別，所以普通流行歌建議手動選擇MAJOR大調
  3. Key (3)
    這是選擇調性的參數，根據伴奏音樂的調來調整，同時也是配合Scale的參數來組成音階
  4. Detune (4)
    預設為440Hz，在鋼琴上是中央a的音，是目前音樂的標準音高，
    如果你有特殊需求，例如某些國樂會用443，就能在這邊調整
  5. Retune Speed (5)
    這是修正音高的速度，單位是ms(毫秒)，時間越短效果器將聲音拉到正確音高的速度越快，但也會越機械化，
    所以要根據歌曲的風格、vocal的轉音來調節需要的速度，使聽感更自然
  6. Vibrato Shape (6)
    Vibrato Pitch (7)
    Vibrato Rate (8)
    這三個是控制顫音，也就是抖音的參數，開啟後會在長音的尾巴加上抖音的效果
    先從Vibrato Shape，將值設為SAWTOOTH或SQUARE或SINE WAVE，三種模式效果不同，
    再從Vibrato Pitch控制抖音的範圍，
    用Vibrato Rate控制抖音的速度，
  7. Input type (11)
    預設是SOPRANO(女高音)，還有其他的中音、低音可選，猜測可能對不同的聲音有做優化
  8. Use Classic Mode DSP (12)
    應該是模擬經典的DSP修音的效果，聽起來的感覺我也不太會形容，可以玩看看
  9. Humanize (30)
    配合前面的參數稍微提高一點這個參數的值，可能會讓聽感更自然，但這個看狀況而定
  10. Learn Scale from MIDI (37)
    可以用這個參數來自訂要讓auto tune自動修調的音符，例如輸入五聲音階的音符，auto tune就會自動將所有的聲音拉到這五個音上，
    詳細的用法在後面會另外說明
    1.Target Notes via MIDI (38)
    如果要用自己編輯的midi音符來指定音高，或實現用midi鍵盤時時控制音高，需要把這個參數的值設為on，後面會介紹如何使用這個功能
  11. Natural Vibrato (63)
    這個參數可以放大或是減小聲音裡原有的抖音，值的範圍是12到-12，
    當值為負數時會抹掉vocal的抖音，但有可能也會影響到小幅度的轉音，
    當值為正數時，則會放大vocal裡的所有抖音
  12. Use Formants (71)
    Throat Length (72)
    這個是調整共振峰的參數，要先將Use Formants設為on，將功能開啟，
    再調整Throat Length的值，就能輕鬆改變vocal的聲音了，
    想了解什麼是共振峰，可以看好和弦的這支影片，在5分40秒左右開始會提到共振峰
    點我看Siri 跟 Google 小姐誰唱歌比較好聽
  13. Transpose (75)
    這個參數可以用來轉key，但要注意這是將修掉後的聲音轉調，並不是改變前面key參數的值，
    此參數的值範圍是12到-12，也就是兩個八度
  14. Latency Removal (92)
    根據網路資料顯示，這個參數可以用來降低處理聲音的延遲，不過我已經是用ASIO了，所以沒有感覺，有需要可以試看看

如何用midi控制音高

  1. 建立一條空軌道，命名為auto-tune，以方便辨識，
    按f打開fx效果視窗，把auto-tune pro掛到軌道上
  2. 在這條軌道上按p，從參數列表找到
    Target Notes via MIDI
    並將值設為on
  3. 按esc退出參數列表，在auto-tune軌道上按快顯鍵，找
    输入: MIDI 子功能表
    往右展開選擇虛擬midi鍵盤或已經接在電腦上的硬體midi鍵盤，再展開選全部通道
  4. 按f7打開auto-tune軌道的錄音準備，之後才能接收從虛擬或硬體midi鍵盤的訊號
  5. 建立一條新軌道，命名為人聲，並在軌道上插入人聲素材
    p.s reaper裡按alt往右找插入，往下找媒體文件enter，選擇要插入的音訊檔
  6. 在人聲軌道上按
    alt+快顯鍵
    往下找發送，往右再往下找auto-tune的軌道enter
  7. 在人聲軌上按alt+快顯鍵，找
    主控发送 勾選
    按enter取消勾選，以避免聲音重複發送到主控軌
  8. 按空格開始播放
  9. 回到auto-tune軌道，按
    alt+b
    來打開虛擬midi鍵盤，或直接在硬體midi鍵盤上彈奏，
    要開始或結束錄音就按r

如何自訂要修調的音符

以上面的無條件為你這首歌為例，key是f#調，
將f#以首調來表示，也就是把f#當作do來說的話，這首歌又用到了升do、升fa跟升si，
但在auto tune pro裡，並沒有提供十二個音的參數給我們用開關來控制，這時就要用到以midi音符來學習的參數，告訴auto tune我們需要的是那些音。

  1. 一樣建立軌道，在上面掛auto tune pro，找到軌道按p，找到這個參數
    Learn Scale from MIDI (37)
    tab一次，往上切成on
  2. 按esc關閉參數視窗，選中這條auto tune的軌道，按快顯鍵，找
    輸入: MIDI 子功能表
    往右展開，選擇要用來輸入音符的虛擬鍵盤或硬體midi鍵盤，再選全部通道enter
  3. 按f7啟動這條軌道的錄音準備
  4. 如果是虛擬midi鍵盤，先按alt+b打開虛擬midi鍵盤
  5. 用nvda加4或6找到
    設置中央C (右鍵單擊想要設置的鍵): 編輯區 C4
    按nvda+小鍵盤enter
    將裡面的自全部刪除，改為你需要的key，例如無條件為你，那就打F#，按enter
    p.s 這裡有個輸入技巧，如果輸入f#或其他#的key，卻跳到b1，可以參考這個作法，
    先打完key，往右動幾下，在輸入#，動幾下，再輸入因層，
    輸入完後用nvda+4或6看一下，如果變成g9，那就再來一次，這次只輸入f跟#，輸入完後看一下，輸入成功有可能會是f#0，ok，這樣就成功了
  6. 按esc關閉虛擬midi鍵盤，再重新打開虛擬midi鍵盤
    輸入f#這個自然大調的音階，從do到si，也就是鍵盤z一直往右到m，接著輸入要額外多出的音，
    以無條件為你這首歌，就再輸入s、g跟j
  7. 輸入完後esc關閉虛擬midi鍵盤，按p打開參數視窗，找到
    Learn Scale from MIDI (37)
    tab一次，往下將值切成off，這樣就設定完成了
  8. 如果前面輸入的音符有問題，或想重新設定音符，只要再將這個參數的值切成on，就能重新輸入音符，
    每次重新開關一次都是新的紀錄，並不會記憶前一次輸入的音符

auto-tune Access修調效果器介紹

如果你已經學會了上文中的使用技巧，用起Access應該是很輕鬆的，他的參數比起auto-tune pro更為簡單，參數也都大同小異，
要做到電音的效果，只要讓Retune Speed縮到最短最快，讓音跟音中間的滑音幾乎消失，那就是電音效果了，
Access還有個好處，我們可以從參數列表看到12個音符，
當選擇了某個key時，歌曲中卻出現了這個key沒有的音，可以透過參數裡來開啟某個音符，使其成為音階的組成音。
最後放上無條件為你的電音版，大家可以聽看看，
這是粗略製作，全部開自動修音的結果，享受吧~
