設定共用資料夾-2 疑難排解篇

 2022.4.29  3 分鐘

設定共用資料夾之疑難排解篇

如果你閱讀了上一篇文章，仍然無法看到共用的資料夾，你可以參考以下的方法來找出問題，
要提醒的是，這些方法都是建立在所有電腦都在相同網段下，如果要連線的電腦屬於不同的網段，還需要其他設定，此篇就不再贅述，
什麼是網段？你可以。
點我查看什麼是網段
點我查看ip地址基本介紹

啟用Windows功能

在我自己測試的時候，就是用這個方法來解決看不到共用資料夾的問題，以下提供參考。

  1. 打開控制台
  2. 在顯示模式為列表的狀態下，找到程式集，按enter進去
    如果你不懂怎麼切換成列表，那你可以回頭看上一篇文章
  3. 按tab找
    開啟或關閉 Windows 功能 連結
    按enter進去
  4. shift+tab找到
    Windows 功能 樹狀檢視
  5. 在這裡按shift或ctrl+shift到英數輸入，並按幾次s，直到聽到
    SMB 1.0/CIFS 檔案共用支援
    在這裡按空格打勾
  6. 接著往右展開這個目錄，往下確定這三個選項都是勾選的
    SMB 1.0/CIFS 用戶端 核取方塊 已選取 勾選
    SMB 1.0/CIFS 自動移除 核取方塊 已選取 勾選
    SMB 1.0/CIFS 伺服器 核取方塊 已選取 勾選
  7. 其他的不要動，按tab找確定，按enter
  8. 此時就會開始安裝Windows功能，安裝完成會提示需要重新啟動電腦，儲存好其他資料後，就可以立即重新啟動
    p.s 要注意的是，所有要進行共用資料夾連線的電腦都要安裝此Windows功能，否則仍然會看不到共用資料夾
  9. 等待電腦重新啟動完成後，測試看看是否可以看得到共用資料夾。

查看工作群組

想讓所有設定好的電腦都能看到共用資料夾，那每台電腦就必須在同一個工作群組，以下說明如何查看以及修改。

  1. 在桌面上找到本機，按快顯鍵，再往上找內容，按enter
  2. 會打開系統內容的視窗，在這按
    nvda+九宮格4或6
    可以找到工作群組，我的是
    WORKGROUP
    基本上沒做任何改動的話預設都是這個，
    如果工作群組不同，要進行修改，
  3. shift+tab找變更設定，按enter
  4. tab找變更(C)... 按鈕
    按空格
  5. 接著打開的視窗內，按tab就能看到電腦名稱以及工作群組的編輯框，修改完成按tab找確定enter完成，之後可能會需要重開機。

設定網路類型

我查閱了網上的資料，建議使用私人的網路類型，
而當連線以及被連線電腦的網路類型不同時，有可能就會連接不上，所以不管設定公用或私人，所有電腦保持一致也是一種方法，避免出問題。

  1. 按win+i打開設定
  2. 按一次tab到列表上，往右找
    網路和網際網路
    按enter
  3. 按幾下tab找到
    網路狀態 群組，變更連線內容 連結
    按enter
  4. 按幾下tab，就能找到公用跟私人的勾選框，建議找到私人並按空格勾選，
    如果你懶的再重新設定看所，那就維持現況，只要將所有電腦的網路類型都保持相同即可，
    如果你進行了修改，那就繼續看下去
  5. 接下來電腦可能會重開機，如果沒有，一樣繼續
  6. 你需要按照上一篇文章的說明，重新將探索和印表機的共用設定保持開啟，以及關閉密碼保護
  7. 完成。

其他注意事項

  1. 確認網路線有插好，不要笑，說不定你真的沒插好，或是連接好wifi，並且網路暢通
  2. 將要連線以及被連線的電腦重開機
  3. 確認控制台的共用設定正確，請參考上一篇的步驟
  4. 確認上面說明的Windows功能已經安裝
  5. 確認網路類型相同，我這裡目前都是設定私人

好，以上就是各種需要注意的地方，如果你還是不行，那只能問Google了，
